API Documentation

Use these endpoints to integrate Factyne into your applications.

POST /api/content/

Submit content for fact-checking.

{
  "raw_text": "COVID-19 vaccines are safe...",
  "url": "https://example.com/article"  // optional
}

Response (201):

{
  "id": 1,
  "trust_score": 0.75,
  "claims_count": 3,
  "contradiction_count": 0
}

GET /api/content/list/

Get all submitted content.

Query params: limit=10, offset=0

GET /api/content/{id}/

Get detailed info about a specific content submission.

GET /api/contradictions/

Get all detected contradictions.

Query params: type=direct_negation, limit=20, offset=0

For more examples, visit the interactive API browser.
